Nel modulo SecureDocument, il file module.configuration.json va configurato nel modo seguente:

	1. Topic_Workflow_Integration -> nome della topic nella quale arrivano gli eventi
	2. Subscription_SecureDocument -> nome della sottoscrizione specifica dalla quale si leggono gli eventi
	3. ProtocolSignature -> formato della segnature del documento utilizzata per gli annessi di protocollo
