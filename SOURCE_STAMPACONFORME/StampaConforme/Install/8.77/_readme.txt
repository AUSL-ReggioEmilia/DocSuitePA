Attenzione! per un corretto funzionamento del servizio converter di OpenOffice è necessario disabilitare gli aggiornamenti automatici di JAVA.
Per farlo, seguire le istruzioni sotto:
 - Andare in Pannello di Controllo > JAVA
 - Tab "Aggiorna"
 - Togliere la spunta dal checkbox "Controlla automaticamente la presenza di aggiornamenti"
